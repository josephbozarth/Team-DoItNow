How to Use APIs:

Send a JSON request with the required data to the URL listed and the server will respond with JSON data or an error message.


Feature Request:
    Create:
        Adds a row to the Feature Request Table
        URL:
            '/api/feature/create'
        Input:
            {
                user_id : int,
                name : string,
                description : string
            }
        Output:
            {
                feature_id : int
            }

    Read By Feature ID:
        Gets a Feature Request Row given the Feature Request ID
        URL:
            '/api/feature/read/featureid'
        Input:
            {
                feature_id : int
            }
        Output:
            {
                feature_id : int,
                user_id : int,
                name : string,
                description : string
            }

    Read By Story ID:
        Gets all of the Feature Request Rows given the User ID
        URL:
            '/api/feature/read/userid'
        Input:
            {
                user_id : int
            }
        Output:
            {
                [
                    {
                        feature_id : int,
                        user_id : int,
                        name : string,
                        description : string
                    },...
                ]
            }

    Update:
        Updates a Feature Request
        URL:
            '/api/feature/update'
        Input:
            {
                feature_id : int
            }
        Output:
            {
                feature_id : int,
                user_id : int,
                name : string,
                description : string
            }

    Delete:
        Deletes a Feature Request
        URL:
            '/api/feature/delete'
        Input:
            {
                feature_id : int
            }
        Output:
            {
                status : string
            }


User:
    Create:
        Adds a row to the User Table
        URL:
            '/api/user/create'
        Input:
            {
                email : string,
                password : string,
                role : string
            }
        Output:
            {
                user_id : int
            }

    Read:
        Gets a User Row given the User ID
        URL:
            '/api/user/read'
        Input:
            {
                email : string
            }
        Output:
            {
                user_id : int,
                email : string,
                role : string
            }

    Update:
        Updates a User
        URL:
            '/api/user/update'
        Input:
            {
                user_id : int,
                email : string,
                password : string,
                role : string
            }
        Output:
            {
                user_id : int,
